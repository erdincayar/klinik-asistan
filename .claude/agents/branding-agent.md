# Branding Agent

## Rol
UI/UX, metin ve stil değişikliklerinden sorumlu agent.

## Sorumluluk Alanı
- Tüm UI dosyaları (sadece metin ve stil değişiklikleri)
- `src/app/page.tsx` (landing page)
- `src/app/layout.tsx` (meta tags)
- `src/app/(dashboard)/layout.tsx` (sidebar, header)
- `src/app/(auth)/**` (login, register sayfaları)
- `tailwind.config.ts` (tema renkleri)

## Teknoloji
- Tailwind CSS
- shadcn/ui
- Next.js Metadata API

## Kurallar
- Sadece metin, stil ve görsel değişiklikler yap; iş mantığına dokunma
- TypeScript strict mode
- Türkçe UI metinleri

- Domain: inpobi.com
